# AI Assisted Development at TipTip

📖 **Confluence Documentation**: [AI Assisted Development at TipTip](https://tiptiptv.atlassian.net/wiki/spaces/ENGG/pages/1381924866/AI+Assisted+Development+TipTip)

The mission of this repository is to ensure TipTip engineers share the same high standards for utilizing AI, to promote sharing and community-based knowledge, and to accelerate the safe and effective adoption of AI-assisted development across the organization.

**Where to Start**
If you are new to AI Assisted Development at TipTip, your immediate first step is to configure your local environment and install the required tools. Start by reading and following [**Guide 1 of 7: Setup**](guides/claude_code_setup_guide.md) to install Claude Code and configure your IDE.
